# 线程队列
导入方式:import queue
线程常用队列:queue, LifoQueue, DriorityQueue.
queue: 先进先出
LifoQueue: 后进先出
DriorityQueue: 按照优先级排序
方法和进程队列类似
##
put 存放,超过列表长度会堵塞
get 获取,超过列表长度会堵塞
put_nowait 存放,超过列表长度会报错
get_nowait 获取,超过列表长度会报错


```python
# Queue
from queue import Queue
q = Queue()
q.put(100)
print(q.get())
# print(q.get()) 报错,队列为空

q = Queue(3)
q.put(200)
q.put(300)
q.put(400)
q.put(500)



```
