# 学习资料分享平台

一个功能完善的学习资料分享平台,支持动态管理百度网盘资料,具备无障碍访问和个人信息管理功能。

## 功能特点

### ✨ 核心功能
- **资料管理**: 添加、编辑、删除学习资料
- **百度网盘集成**: 专门针对百度网盘链接优化
- **分类管理**: 支持多种分类(编程、Python、Java、网络安全、设计、语言、工具、其他)
- **高中资料**: 独立的高中资料分类,包含语数外物化生学科子分类
- **标签系统**: 为资料添加标签,方便检索
- **搜索功能**: 实时搜索资料名称、描述和标签
- **浏览统计**: 记录资料浏览次数,识别热门资料

### 🎨 界面设计
- 复古北大风格主题(深红色+金色)
- 左侧导航栏快速切换页面
- 卡片式资料展示
- 响应式布局,支持手机和电脑
- 流畅的动画效果

### ♿ 无障碍功能
- **字体大小调节**: 小/正常/大三种字体大小
- **高对比度模式**: 黑白高对比度显示
- **字间距调节**: 自定义字间距
- **行高调节**: 自定义行高
- **键盘导航**: 支持快捷键操作
  - `Alt + 1`: 资料库
  - `Alt + 2`: 个人信息
  - `Alt + 3`: 系统设置
  - `Alt + A`: 无障碍模式
  - `Alt + M`: 切换侧边栏

### 👤 个人信息管理
- 基本信息管理(用户名、邮箱、手机、简介)
- 学习信息管理(学校、年级、学习兴趣)
- 个人统计(分享资料数、总浏览量、使用天数)

### ⚙️ 系统设置
- 显示设置(每页显示数量、排序方式)
- 无障碍功能开关
- 数据导出/导入(JSON格式备份)
- 数据清空功能

### 💾 数据存储
- 使用 localStorage 本地存储
- 数据持久化保存
- 支持数据导出备份
- 支持数据导入恢复

## 使用方法

### 1. 直接使用
直接用浏览器打开 `index.html` 即可使用

### 2. 导航使用
- **资料库**: 浏览和管理所有学习资料
- **个人信息**: 管理您的个人资料和查看统计数据
- **系统设置**: 自定义系统配置和数据管理

### 3. 添加资料
1. 点击"➕ 添加资料"按钮
2. 填写资料信息:
   - 资料名称 (必填)
   - 选择分类 (必填)
   - 百度网盘链接 (必填)
   - 提取码 (可选)
   - 资料描述 (可选)
   - 标签 (可选,用逗号分隔)
3. 点击"保存"

### 4. 高中资料
1. 选择"高中资料"分类
2. 选择学科(语文/数学/英语/物理/化学/生物)
3. 高中资料会显示学科标签

### 5. 无障碍模式
- 点击左侧导航栏"♿ 无障碍模式"按钮
- 或使用快捷键 `Alt + A`
- 可调节字体大小、对比度、字间距、行高等

### 6. 数据备份与恢复
- **导出**: 设置页面 → 数据管理 → 导出所有资料
- **导入**: 设置页面 → 数据管理 → 导入资料

- Chrome (推荐)

- Safari

## 注意事项

1. 数据存储在浏览器本地,清除浏览器数据会丢失资料
2. 建议定期使用"导出数据"功能备份资料
3. 百度网盘链接需要符合格式: `https://pan.baidu.com/s/...`
4. 无障碍模式设置会保存,下次访问自动应用
