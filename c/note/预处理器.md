## 工作原理及指令
预处理器是以"**#**"开头的,并且必须是第一个字符. \
比较重要的预处理器指令: 
```txt
#define 定义宏
#include 包含一个源代码文件
#undef 取消已定义的宏
#ifdef 如果宏已经定义,则返回真
#ifndef 如果宏没有定义,则返回真
#if 如果给定条件为真,这编译下面代码
#else #if的替代方案
#elif 如果前面的#if给定条件不为真,当前条件为真,则编译下面代码
#endif 结束一个#if...#else条件编译块
#error 当遇到标准错误时,输出错误信息
#pragma 使用标准化方法,向编译器发布特殊的命令到编译器中
```
下面通过实例说明:
```c
#define Word 1
```
这个指令告诉预处理器把所有的Word替换为1
```c
#include <plan A>
#include "mywork"
```
以上指令代表将系统库中的<plan A>,添加到源文件中,并且下面需要从本地目录中获取"mywork"文档,同样添加到源文件中.
```c
#undef Plan B
#define Plan B 100
```
以上指令含义为取消plan B,并且将其定义为100
```c
#ifndef PLAN C
#define PLAN C "D!"
#endif
```
以上指令代表只有PLAN C没有被定义时,才可以去定义D
