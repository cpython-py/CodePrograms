## 函数
每个完成的程序都必须至少含有一个函数,也就是**主函数main()** \
在C语言中一个完整的函数包括函数头和函数主体,其所组成部分包括函数 \
主体,函数名称,参数,返回类型. \
### 函数声明
返回类型 函数名称(参数); \
如: 
```c
int main(int A, int B);
```
需要注意的是一个函数中参数的名称是次要的, \
而参数的类型是最重要的, \
当我们忽略参数名称,可以发现,函数运行还是正确的. \
如:
```c
int main(int, int);
```
### 调用函数
#### 1. 静态函数
如果一个函数只能被本文件中的函数所调用,那么这个函数就是**内部函数** \
在定义内部函数时,前面必须加上关键字**static** \
如:
```c
static int max(int a, int b);
```
这被称为**静态函数**
#### 2. 外部函数
如果一个函数可以被其他函数所调用,那么这个函数就是**外部函数** \
在函数的左端要加上关键字**extern** \
如:
```c
extern int max(int 1, int 2);
```
当然**extern**在定义函数时可以不加,这种情况被默认为外部函数
### 函数参数
#### 调用函数的方法有三种
##### 1. 传值调用
将参数的实际值赋予到形参上,并且形参不会受到实参的影响 \
如:
```c
#include <stdio.h>
void swap(int c, int d);


void swap(int c, int d){
    int temp;
    temp = c;
    c = d;
    c =temp;
}

int main(int argc, char *argv[]){
    int a = 5;
    int b = 10;
    swap(a, b); //调用交换函数
    printf("交换结果为a=%d,b=%d\n",a,b);
    return 0;
}
```
##### 2. 引用调用
通过引用的方式将形参指向实参,这是对形参的操作也会反应的实参上, \
并对其进行同样的操作 \
如:
```c
#include <stdio.h>
void swap(int &x, int &y);


void swap(int &x, int &y){
    int temp;
    temp = x;
    x = y;
    y = temp;
}

int main(int argc, char *argv[]){
    int a = 1;
    int b = 2;
    swap(a, b); //调用交换函数
    printf("交换结果为a = %d,b=%d\n",a,b)
}
```
##### 3. 指针传递
用指针的方式,将两个或者更多的参数传递给另外两个或更多参数的过程
如:
```c
#include <stdio.h>
void swap(int *x, int *y);


void swap(int *x, int *y){
    int temp;
    temp = *x;
    *x = *y;
    *y = temp;
}

int main(int argc, char *argv[]){
    int a = 1;
    int b = 2;
    swap(&a, &b); //调用交换函数
    printf("交换结果为a=%d,b=%d\n",a,b);
    return 0;
}
```
