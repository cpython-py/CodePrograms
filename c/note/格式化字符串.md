## 格式化字符串
```txt
\\ -> \字符
\" -> "字符
\? -> ?字符
\a -> 警报铃声
\b -> 退格键
\f -> 换页符
\n -> 换行符
\r -> 回车
\ooo -> 一到三位的八进制数
\xhh... -> 一个或多个数字的十六进制数
```
