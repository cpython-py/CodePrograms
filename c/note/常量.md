## 常量
### 定义常量
c语言中定义常量分为预处理器定义和关键字定义两种
#### 1. **define**预处理器
如:
```c
#define identifier value
```
如:
```c
#include <stdio.h>


#define LENGTH 5
#define WIDTH 10
#define NEWLINE '\n'


int main(){
    int area;
    area = LENGTH * WIDTH;
    printf("value of area: %d", area);
    printf("%c", NEWLINE);

    return 0;
}
```
#### 2. **const**关键字
我们也可以使用const前缀来指定变量:
```c
const type variable = value;
```
如:
```c
#include <stdio.h>


int main(){
    const int LENGTH = 5;
    const int WIDTH = 10;
    const char NEWLINE = '\n';
    int area;


    area = LENGTH * WIDTH;
    printf("value of area: %d", area);
    printf("%c", NEWLINE);


    return 0;
}
```
其结果如下: 
```txt
value of area: 50
```

八进制前缀:0 \
十六进制前缀: 0x 或 0X \
如: 
```txt
027 -> 八进制
0xaf -> 十六进制
```

无符号整数和长整数为后缀
无符号整数:U \
长整数:L \
如:
```txt
10u -> 无符号整数
10l -> 长整数
10ul -> 无符号长整数
```
