# Change Log

All notable changes to the "Context Menu Manager" extension will be documented in this file.

## [1.0.0] - 2026-03-20

### Added
- 🎉 Initial release
- ✅ Dynamic menu item management (add, edit, delete)
- ✅ Support for editor and explorer context menus
- ✅ Custom SVG icon support with beautiful styling
- ✅ Keyboard shortcut configuration
- ✅ Conditional visibility based on:
  - File types
  - Project paths
  - Selection state
  - Workspace trust
  - File count
- ✅ Multiple execution types:
  - VS Code commands
  - Scripts
  - Terminal commands
  - URLs
- ✅ Visual configuration panel with WebView
- ✅ Sidebar menu items view
- ✅ Export/Import configuration
- ✅ Configuration backup and restore
- ✅ Reset to default functionality
- ✅ Beautiful SVG icons for all operations
- ✅ Detailed logging and error handling
- ✅ Comprehensive documentation

### Features
- 🎨 Beautiful and modern UI design
- 🚀 High performance and low memory usage
- 💾 Persistent configuration storage
- 🔄 Automatic state synchronization
- 📝 Full TypeScript support
- 🧪 Extensible architecture

### Technical
- Written in TypeScript
